# Gemini Chat Analyze & Evolver

## 🌟 项目愿景
这是一个专为开发者设计的**“人机协作进化系统”**。它不仅是一个对话查看器，更是一个通过分析历史行为来优化开发者表达能力、发现潜在需求、并自动化工作流的深度分析平台。

## 🎯 核心目标
1. **可视化现状**: 精美地展示 Gemini CLI 的对话历史、思考链和工具调用轨迹。
2. **行为画像**: 自动识别工作范畴（开发、学习、运维等），分析操作习惯。
3. **表达教练**: 审计指令的精准度，提供“提问语言艺术”的优化建议。
4. **Skill 孵化**: 将高频重复的操作序列转化为 Gemini CLI 的自定义 Skills。
5. **模型审计**: 追踪不同模型（Pro vs Flash）在特定任务下的表现差异。
