# 📋 详细需求清单

## 1. 基础观测 (Viewer)

- **项目树导航**: 识别 `~/.gemini/tmp/` 下的 Project Slug。
- **富文本会话**: 支持 Markdown、代码高亮、Thoughts (思考链) 折叠展示。
- **工具流追踪**: 展开查看每个工具调用的 `Args` 和 `Result`。
- **快照集成**: 在时间轴上标记 `checkpoint-*.json` 的位置。

## 2. 行为模式分析 (Analyst)

- **任务分类**: 自动识别 Coding (开发)、Study (学习)、Ops (运维)、Arch (架构) 等任务类型。
- **轨迹识别**: 提取工具调用链（如 `grep` -> `read` -> `edit`），发现重复模式。
- **效率评估**: 统计对话轮数、纠错次数及任务达成率。

## 3. 表达进化教练 (Coach)

- **精准度审计**: 识别模糊词（如“那个”、“这里”）及缺失的上下文。
- **语言艺术优化**: 为历史提问提供更专业、更具架构师风格的改写建议。
- **成长曲线**: 可视化展示用户指令精准度的提升过程。

## 4. 模型与技能审计 (Audit & Evolution)

- **模型对比**: 统计同一任务/Skill 在不同 LLM 模型下的成功率。
- **Skill 孵化**: 选中成功对话路径，一键导出为 `SKILL.md` 模板。
- **健壮性监控**: 监控 Skill 的“健康度”，提示模型选型建议。

## 5. 技术特性

- **实时同步**: 监听本地文件系统变动，Web 端准实时刷新。
- **离线优先**: 数据完全存储在本地 `~/.gemini/tmp`。
- **全量搜索**: 支持跨项目、跨会话的语义搜索。
